# Sprint 03 — Git Service + Jobs

## Цель
Подключить git CLI, job execution и автоматическое обновление state после git-операций.

## Exit Criteria
Host умеет запускать job ops v0.1 и корректно обновлять snapshots после выполнения.

## Правила выполнения
- задачи можно вести параллельно только если явно нет зависимостей
- если task меняет контракт, нужно синхронно обновить docs
- каждый task должен закончиться проверяемым артефактом
