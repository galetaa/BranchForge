# Сделать безопасный runner для git CLI

## Контекст
Нужен единый путь запуска git-команд без shell.

## Цель
Есть единый исполнитель git процессов.

## Подзадачи
1. Реализовать process launcher.
2. Собирать stdout/stderr/exit_code.
3. Пробрасывать cwd.
4. Нормализовать ошибки в E_GIT_ERROR.

## Зависимости
- нет

## Артефакты
- изменения в коде
- тесты на заявленный уровень
- обновление документации, если менялся контракт/архитектура

## Acceptance Criteria
- Runner покрыт unit tests.
- Нет shell invocation.

## Definition of Done напоминание
- код собран
- тесты проходят
- документация синхронизирована
- сценарий задачи воспроизводим
