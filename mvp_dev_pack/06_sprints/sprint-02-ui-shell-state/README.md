# Sprint 02 — UI Shell + State

## Цель
Поднять host UI, command palette, store и host-rendered ViewModel.

## Exit Criteria
Host показывает окно, палитру команд, умеет хранить snapshots и рендерить status panel из viewmodel.

## Правила выполнения
- задачи можно вести параллельно только если явно нет зависимостей
- если task меняет контракт, нужно синхронно обновить docs
- каждый task должен закончиться проверяемым артефактом
