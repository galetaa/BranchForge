# Сделать базовый layout окна и view slots

## Контекст
Нужна визуальная основа для дальнейшего demo-сценария.

## Цель
Host умеет размещать host-rendered views.

## Подзадачи
1. Поднять базовое окно.
2. Сделать left slot.
3. Сделать служебную область для palette/status сообщений.
4. Определить состояние активного view instance.

## Зависимости
- нет

## Артефакты
