# Sprint 04 — Repo Manager Plugin

## Цель
Собрать пользовательский сценарий открытия репозитория через plugin-first flow.

## Exit Criteria
Пользователь через command palette открывает репозиторий и видит, что host/store/plugins синхронно переходят в repo-opened состояние.

## Правила выполнения
- задачи можно вести параллельно только если явно нет зависимостей
- если task меняет контракт, нужно синхронно обновить docs
- каждый task должен закончиться проверяемым артефактом
