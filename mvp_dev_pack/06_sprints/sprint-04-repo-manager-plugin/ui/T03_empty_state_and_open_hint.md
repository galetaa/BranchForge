# Сделать пустое состояние до открытия repo

## Контекст
Приложение должно быть понятным до первого действия пользователя.

## Цель
Стартовый UX не выглядит сломанным.

## Подзадачи
1. Показать empty state при `repo.is_open=false`.
2. Добавить явную подсказку про `Open Repository`.
3. Сделать graceful switch после открытия repo.

## Зависимости
- ../sprint-02-ui-shell-state/ui-shell/T01_window_layout_and_slots.md

## Артефакты
- изменения в коде
- тесты на заявленный уровень
- обновление документации, если менялся контракт/архитектура

## Acceptance Criteria
- Пустое состояние присутствует и не конфликтует с palette.
- После успешного repo.open UI переключается на status panel без перезапуска.

## Definition of Done напоминание
- код собран
- тесты проходят
- документация синхронизирована
