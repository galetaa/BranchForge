Показать Status panel и реализовать working tree workflow stage/unstage.

## Exit Criteria
Status plugin рендерит staged/unstaged/untracked списки, selection работает, stage/unstage выполняются и обновляют UI.

- если task меняет контракт, нужно синхронно обновить docs
- каждый task должен закончиться проверяемым артефактом
